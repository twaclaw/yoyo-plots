# Yo-Yo-Math Plots

> A collection of utils generating some of the figures in the  [Yo-Yo-Math book](...).

![](./cover.png)


## Disclaimer

I am publishing this as a separate repository in the hope that it will be useful to someone. Please note that some of the code, particularly the boilerplate, was co-generated by an LLM. I tried to sanitize it as much as possible, though not exhaustively.


## Installing

```bash
git clone https://github.com/twaclaw/yoyo-plots
cd yoyo-plots
uv venv --python=3.13 # or a different supported version
uv sync
```

## Examples

Check [examples.ipynb](./examples.ipynb) for some examples.


Additional examples can be found in the source files of [yo-yo-math](https://github.com/twaclaw/yo-yo-math).
